🚀 UniGame - Space Shooter

Jogo de naves espaciais feito em Python (Pygame). Controle sua nave, destrua inimigos e acumule pontos.

▶️ Como rodar
git clone https://github.com/raphafg/UniGame.git
cd UniGame
pip install -r requirements.txt
python main.py

🎮 Controles
← / → mover a nave
Espaço atirar
Esc sair

🛠️ Tecnologias
Python 3
Pygame
